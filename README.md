# C语言围棋游戏

这是一个使用C语言开发的围棋游戏，具有图形界面，支持人机对战、悔棋、复盘等功能。

## 功能特点

- **完整的围棋规则实现**：包括气的判断、提子规则、劫争规则等
- **图形化界面**：使用SDL2库实现直观的游戏界面
- **人机对战**：内置简单AI，可与电脑对弈
- **棋局回溯系统**：支持悔棋、重做和复盘功能
- **模块化设计**：代码结构清晰，便于理解和扩展

## 项目结构

- `main.c` - 主程序入口
- `go_game.h/go_game.c` - 围棋游戏核心逻辑
- `go_ui.h/go_ui.c` - 图形界面实现
- `Makefile` - 项目构建文件

## 编译和运行

### 依赖项

本项目依赖以下库：

- SDL2
- SDL2_ttf

### 在Windows上编译

1. 安装MinGW和MSYS（或其他C编译环境）
2. 安装SDL2和SDL2_ttf开发库
3. 使用以下命令编译：

```bash
make
```

或者手动编译：

```bash
gcc -o go_game main.c go_game.c go_ui.c -lSDL2 -lSDL2_ttf -lm
```

### 运行游戏

```bash
./go_game
```

## 游戏操作

- **鼠标左键**：在棋盘上落子
- **U键**：悔棋
- **R键**：重做
- **P键**：复盘
- **A键**：AI落子（电脑下棋）
- **ESC键**：退出游戏

## 技术实现

### 气的判断算法

使用广度优先搜索（BFS）算法寻找棋子周围的空地，计算气的数量。当一组棋子的气为0时，这些棋子会被提走。

### 棋局回溯系统

使用双向链表记录每一步落子的位置和颜色，支持悔棋和复盘功能。

### 劫争规则

记录最后一次提子的位置，如果只提了一颗子，则不允许立即在该位置落子，避免无限循环。
