# 围棋游戏 (Go Game)

基于SDL图形库开发的围棋游戏，实现了围棋的基本规则和人机对弈功能。
框架由AstreoX搭建，MCTS参数由国科大同学BigTurtle调试

## 功能特点

### 基本功能
- 使用SDL图形化界面创建19×19的棋盘
- 实现围棋基本规则（自动提子、气的判断、自杀行为判断）
- 悔棋和回溯棋局功能（使用双向链表存储）
- 实时计算黑白双方气数并判断胜负

### 进阶功能
- 基于蒙特卡洛树搜索(MCTS)的AI对弈功能
- 打劫行为判断与提示

## 编译和运行

### 依赖项
- SDL2
- SDL2_image

### 编译
```
make
```

### 运行
```
./cgogame
```

## 操作说明
- [A] - 切换AI模式
- [U] - 悔棋
- [P] - 回溯棋局
- [T] - 显示/隐藏提示
